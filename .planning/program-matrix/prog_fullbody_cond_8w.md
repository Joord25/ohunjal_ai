# prog_fullbody_cond_8w — 풀바디 컨디셔닝 8주 매트릭스

**생성:** 2026-04-30 (회의 ζ-5 / SEED-002 P2)
**target:** endurance goal — 전신 균형 + 컨디셔닝 (체력·근지구력·work capacity)

## 분할 — 풀바디 3일 (Day A / Day B 교대)

### Day A (Squat-focused full body)
| 슬롯 | 자극 | 유사 운동 그룹 |
|---|---|---|
| Slot 1 (스쿼트 compound) | 무릎 dominant | 백 스쿼트 / 프론트 스쿼트 / 고블렛 스쿼트 |
| Slot 2 (Push compound) | 가슴 horizontal | 벤치 / 인클라인 / 푸쉬업 |
| Slot 3 (Pull compound) | 등 horizontal | 시티드 로우 / 인버티드 로우 / 벤트오버 로우 |
| Slot 4 (코어 + finisher) | 코어 + 컨디셔닝 | 플랭크 / 행잉 니레이즈 + 30s 케틀벨 스윙 |

### Day B (Hinge-focused full body)
| 슬롯 | 자극 | 유사 운동 그룹 |
|---|---|---|
| Slot 1 (힌지 compound) | 힌지 | 컨벤셔널 데드 / RDL / 트랩바 데드 |
| Slot 2 (Push vertical) | 어깨 vertical | OHP / 머신 숄더 프레스 / 푸쉬업 + 어깨 강조 |
| Slot 3 (Pull vertical) | 등 vertical | 풀업 (어시스티드) / 랫 풀다운 |
| Slot 4 (단일다리 + 코어) | 단일다리 + 코어 | 워킹 런지 + 사이드 플랭크 |

## Chapter 매트릭스

### Chapter 1 — Adaptation (W1-W4) "30분 풀바디"
| 항목 | 값 |
|---|---|
| Sets × Reps | 3 × 12-15 |
| RPE | 7 |
| 세션 길이 | 30-35분 |
| Finisher | 1세트 30s (Slot 4 마지막) |
| Progression | 매주 +1 rep/set 또는 무게 +2.5kg |

### Chapter 2 — Build (W5-W8) "45분 풀바디 + 컨디셔닝 ↑"
| 항목 | 값 |
|---|---|
| Sets × Reps | 3-4 × 12-20 (점진 ↑, endurance 강조) |
| RPE | 7-8 |
| 세션 길이 | 40-50분 |
| Finisher | **3 라운드 30s on / 30s off** (케틀벨 스윙 / 버피 / 점프 스쿼트) |
| Progression | 변형 더 어렵게 (예: 푸쉬업 → 디클라인 → 박수) |
