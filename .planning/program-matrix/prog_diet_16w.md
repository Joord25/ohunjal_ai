## 4 Chapter 매트릭스 (16주 = 4w × 4)

### Chapter 1 — Base (W1-W4) "적응 + 폼"
| 항목 | 값 |
|---|---|
| Sets × Reps | 3 × 12-15 |
| RPE | 7 |
| 1RM% | 60-65% |
| 카디오 | LISS 20분 × 2/주 |
| 적자 | -15% kcal (보수적, 자연 보디빌더 0.5% bw/주) |
| Diet break | W4 종료 시 1주 maintenance (선택) |

### Chapter 2 — Build (W5-W8) "Volume ↑"
| 항목 | 값 |
|---|---|
| Sets × Reps | 3-4 × 10-12 |
| RPE | 8 |
| 1RM% | 65-75% |
| 카디오 | LISS 25분 × 2 + HIIT 1 |
| 적자 | -15-20% kcal |
| Diet break | W8 종료 시 1주 maintenance 권장 (Norton refeed) |

### Chapter 3 — Peak1 (W9-W12) "Intensity ↑ + finisher"
| 항목 | 값 |
|---|---|
| Sets × Reps | 4 × 8-10 + drop set |
| RPE | 8-9 |
| 1RM% | 75-80% |
| 카디오 | HIIT 2 + LISS 1 |
| 적자 | -20% kcal (마무리 본격) |
| Diet break | W12 종료 시 carb refeed 3-14일 (필수, Norton + Macroperiodization) |

### Chapter 4 — Peak2 (W13-W16) "최종 마무리"
| 항목 | 값 |
|---|---|
| Sets × Reps | 4 × 6-10 (피라미드) | 
| RPE | 9 (마지막 세트 만) |
| 1RM% | 75-85% |
| 카디오 | HIIT 2-3 + 일일 걸음 12-15K |
| 적자 | -20-25% kcal (단기 큰 적자, Israetel TIA 차용) |
| Diet break | W16 종료 후 = 본격 maintenance reverse diet 시작 |

## prog_summer_diet_12w 와 차이 (16주 vs 12주)

| 항목 | summer_diet_12w | **diet_16w** |
|---|---|---|
| 기간 | 12주 (3 chapter) | **16주 (4 chapter)** |
| 적자 | -20% (단계 X) | **-15% → -25% 점진** |
| Diet break | W5 선택 | **W4·W8·W12·W16 명시 (자연 보디빌더 표준)** |
| 적용 페르소나 | 30대 직장 다이어트 | **본격 자연 보디빌더 또는 4개월 deep cut 결심자** |
| 강도 1RM% | 60-80% | **60-85%** |
| 카디오 | LISS·HIIT 분기 | **HIIT 비중 ↑ (Peak1·Peak2)** |
| Macrocycle 위치 | Standalone | **단독 또는 muscle_8w/inbody_d_12w 후속** |
