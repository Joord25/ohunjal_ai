## 슬롯 정의

### `arms_main_1` (이두·삼두 균형, 6 슬롯)
| 슬롯 | 자극 | 유사 운동 그룹 |
|---|---|---|
| Slot 1 (이두 compound) | 이두 | 바벨 컬 / EZ 바 컬 / 친업 |
| Slot 2 (삼두 compound) | 삼두 | 클로즈그립 벤치 / 딥 / 푸쉬다운 |
| Slot 3 (이두 isolation) | 이두 brachialis | 해머 컬 / 케이블 컬 |
| Slot 4 (삼두 isolation) | 삼두 long head | 오버헤드 익스텐션 / 스컬크러셔 |
| Slot 5 (전완) | 전완 + 그립 | 리버스 컬 / 데드 행 |
| Slot 6 (펌프 finisher) | 펌프 | 21s (이두) + 21s (삼두) — 7+7+7 reps × 2 |

### `arms_bicep_focus` / `arms_tricep_focus`
한쪽 강조 (4-5 슬롯) + 다른 쪽 보조 (1-2 슬롯).

### `arms_pump_finisher` (D6, 15-20분)
- 5분 AMRAP: 컬 + 푸쉬다운 super set
- 5분 펌프: 케이블 컬 + 케이블 푸쉬다운 (드롭셋)
- 5분 최종 팔 펌프
