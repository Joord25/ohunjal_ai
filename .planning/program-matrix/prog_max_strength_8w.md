# prog_max_strength_8w — 최대근력 8주 매트릭스

**생성:** 2026-04-30 (회의 ζ-5 / SEED-002 P2)
**target:** muscle_gain advanced — 1RM 갱신 / 6개월+ 헬스 경험자 (`minExperienceMonths: 6`)
**기간:** 8주 (4주 × 2 chapter)
**주당 빈도:** 4회 weight + 보조 카디오 1-2회 (선택)

## 분할 — 4 lifts × 4일 (Wendler 5/3/1 표준)

| 요일 | 메인 lift | 보조 |
|---|---|---|
| 월 | OHP (Overhead Press) | Pull (풀업/로우) |
| 화 | Deadlift | Squat 보조 (런지/레그 컬) |
| 목 | Bench Press | Pull (시티드 로우/페이스풀) |
| 금 | Squat | Hinge 보조 (RDL/글루트) |

각 day = 메인 lift 1개 (5/3/1 wave) + 보조 4-5 슬롯 (8-12회 RPE 7).

## 8주 매트릭스 — Wendler 5/3/1 × 2 cycle

### Cycle 1 (W1-W4)
| Week | Wave | 메인 lift sets | 비고 |
|---|---|---|---|
| W1 | A | 75%×5 / 80%×5 / 85%×5+ (마지막 AMRAP) | Training Max (TM) = 1RM × 90% |
| W2 | B | 80%×3 / 85%×3 / 90%×3+ | — |
| W3 | C | 75%×5 / 85%×3 / 95%×1+ | Peak intensity |
| W4 | Deload | 60%×5 / 65%×5 / 70%×5 | 회복 |

### Cycle 2 (W5-W8) — Training Max +5lb (upper) / +10lb (lower)
| Week | Wave | 메인 lift sets | 비고 |
|---|---|---|---|
| W5 | A | 새 TM 기준 75%×5 / 80%×5 / 85%×5+ | Cycle 1 보다 무거움 |
| W6 | B | 80%×3 / 85%×3 / 90%×3+ | — |
| W7 | C | 75%×5 / 85%×3 / 95%×1+ | **Peak — PR 시도** |
| W8 | Deload | 60-70% | 회복 + 이후 1RM 테스트 |

## 슬롯 (보조, 4-5 슬롯)

각 day 메인 lift 후 보조 운동 = 일반 슬롯 (Push/Pull/Squat/Hinge 패턴, 앞 카탈로그와 동일).
보조 RPE 7 / 8-12회 (compound 메인은 RPE 9 / 3-5회).
