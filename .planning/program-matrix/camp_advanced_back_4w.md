## 슬롯 정의

### 월 — `back_thickness` (5 슬롯, horizontal pull)
| 슬롯 | 자극 | 유사 운동 그룹 |
|---|---|---|
| Slot 1 | 컴파운드 horizontal pull | 벤트오버 바벨 로우 / T-바 로우 / 펜드레이 로우 |
| Slot 2 | 케이블 horizontal pull | 시티드 케이블 로우 (와이드/클로즈) / 체스트 서포티드 로우 |
| Slot 3 | 등 두께 isolation | 덤벨 로우 (한쪽씩) / 머신 로우 |
| Slot 4 | 후면 어깨 + 트랩 | 페이스풀 / 슈러그 / 리어 델트 플라이 |
| Slot 5 | 등 hold (등척, 30s) | 데드 행 / 인버티드 로우 hold |

### 수 — `back_width` (5 슬롯, vertical pull + lat 강조)
| 슬롯 | 자극 | 유사 운동 그룹 |
|---|---|---|
| Slot 1 | 풀업 (가능 시) | 풀업 / 친업 / 어시스티드 풀업 |
| Slot 2 | 랫 풀다운 변형 | 랫 풀다운 (와이드 그립) / 머신 풀다운 / 클로즈그립 풀다운 |
| Slot 3 | 풀오버 (lat 분리) | 케이블 풀오버 (스트레이트암) / 덤벨 풀오버 / 머신 풀오버 |
| Slot 4 | 등 isolation | 어시스티드 친업 (lat 강조) |
| Slot 5 | 데드 행 (등척, 30-45s) | 데드 행 / 행잉 사이드 로우 hold |

### 금 — `back_volume` (5 슬롯, 펌프 + 마무리)
| 슬롯 | 자극 | 유사 운동 그룹 |
|---|---|---|
| Slot 1 | 컴파운드 (가벼운, 고반복) | 시티드 로우 / T-바 로우 (15회) |
| Slot 2 | 후면 어깨 펌프 | 페이스풀 / 리어 델트 플라이 (15-20회) |
| Slot 3 | 이두 (등 super set 가능) | 바벨 컬 / 해머 컬 |
| Slot 4 | 트랩 + 어깨 | 슈러그 / 업라이트 로우 |
| Slot 5 | **펌프 finisher** (드롭셋) | 케이블 로우 드롭셋 (3 weight) |

## 4주 매트릭스 (Block 1 단축, Nippard 패턴)

| Week | Sets × Reps | RPE | 비고 |
|---|---|---|---|
| W1 | 3 × 8-12 | 7 | 적응 (Block 1 entry, Nippard) |
| W2 | 4 × 8-12 | 7-8 | volume ↑ |
| W3 | 4 × 6-10 | 8 | 강도 ↑ |
| W4 | 4 × 6-10 + drop set 마지막 | 8-9 | finisher 도입 |
