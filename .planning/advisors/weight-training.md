# 웨이트 / 근력 자문단

**용도**: 웨이트 프로그램 설계 (balanced/split/home), 강도·볼륨 규칙, 부상 예방, 체력 평가 기준

**구성**: 사내/국내 포지션 중심. 추후 해외 실존 전문가(Greg Nuckols, Brad Schoenfeld, Mike Israetel 등) 편입 검토 예정.

## 현 편성

| 역할 | 전문성 | 소환 기준 |
|------|--------|----------|
| 국가대표 운동코치 (20년) | 프로그래밍, 주기화, 고강도 설계 | 운동 프로그램 설계, 강도/볼륨 변경 |
| 재활스포츠의학 교수 | 부상 방지, 안전 가이드라인 | 운동 안전, 컨디션 체크 로직 |
| 한체대 교수 | 운동생리학, 체력 평가 기준 | 체력 테스트, 등급 기준 변경 |
| 운동생리학자 | ACSM/NSCA 가이드라인 | 강도 분류, 부하 계산 |
| 건강운동관리사 (15년) | 일반인 운동 처방 | 초보자 프로그램, 안전 범위 |
| 물리치료사 (20년+) | 재활, 움직임 평가 | 스트레칭/워밍업, 부상 예방 |
| 대표 (임주용) | 트레이너 실무, 도메인 최종 감각 | 모든 회의 상시 참석 |

## 소환 원칙

- 강도·볼륨 변경 → 국가대표 코치 + 운동생리학자
- 초보자 프로그램 (home/guest) → 건강운동관리사 + 재활스포츠의학 교수
- 새 운동 추가/제외 → 국가대표 코치 + 물리치료사
- 평가 기준 (e1RM, bw ratio 등) → 한체대 교수 + 운동생리학자
