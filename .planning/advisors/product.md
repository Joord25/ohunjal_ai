# 프로덕트 / 디자인 / 콘텐츠 자문단

**용도**: UI 변경, 새 화면, 바텀시트/모달, 카피·톤앤매너, 다국어 뉘앙스, 랜딩 SEO, 구독 플로우·온보딩·CTA

| 역할 | 전문성 | 소환 기준 |
|------|--------|----------|
| UX/UI 디자이너 | 인터페이스 설계, 인터랙션 패턴 | UI 변경, 새 화면, 바텀시트/모달 |
| 콘텐츠 MD | 유저 대면 텍스트, 톤앤매너 | 코치 멘트, 메시지 톤, 문구 변경 |
| 카피라이터 | 마케팅 카피, 랜딩 텍스트 | 랜딩 페이지, 구독 화면 문구 |
| 현지화 전문가 | i18n, 언어별 뉘앙스 | 다국어 번역, locale별 톤 조정 |
| SEO 전문가 | 검색 최적화, 메타 태그 | 랜딩 SEO, 메타 description |
| 그로스 마케터 | 전환율, 리텐션, 퍼널 | 구독 플로우, 온보딩, CTA |

## 소환 원칙

- **UI 신규 화면/컴포넌트** → 디자이너 + 그로스 마케터 (전환율 관점)
- **코치 멘트·시스템 문구** → 콘텐츠 MD (+ 프롬프트 엔지니어링 자문단 병행)
- **랜딩·구독 카피** → 카피라이터 + 그로스 마케터
- **다국어 추가** → 현지화 전문가 (메모리 `feedback_llm_data_sanitization.md` 참조)
- **SEO** → SEO 전문가 (메모리 `feedback_seo.md` 참조 — 자연어 문장형, 구분자 미사용)
- **러닝 프로그램 UI** → 디자이너 + 콘텐츠 MD (프로그램 선택 화면, 주차 진행률 UI, 세션 카드)
